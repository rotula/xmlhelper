********************
README: XML Services
********************

Provide some basic xml services that are often needed for text
processing with lxml.

As of November 4, 2024 (with the release of version 0.22.2, the last
one, that is guaranteed to be compatible with Python 2), support for
Python 2 has officially been dropped.
